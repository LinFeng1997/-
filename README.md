## 启动这个项目

* 环境：node
* UI框架：ionic3
* JS框架：Angular4+Typescript
* 连接设备API：cordova
* 本地存储：pouchDB
* 工具库：lodash/chartjs/highchart/moment等

### 项目结构


### ionic安装

```bash
$ npm install -g ionic
```

### 运行

```bash
$ ionic serve
```
